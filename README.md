# 🗺️ 迷雾大陆全自动看广告领奖励脚本 (mwdl-autoad)

本脚本是一个基于 **Python + uiautomator2 + OpenCV** 的 Android 游戏自动化脚本，专为《迷雾大陆》（小游戏/手游端）设计，可实现全自动循环“**截屏检测 -> 识别广告按钮 -> 点击进入 -> 轮询等待播放 -> 自动识别并关闭广告 -> 确认领取 -> 循环下一页**”的自动化流程。

---

## ✨ 核心特性

1.  **多尺度模板匹配 (Multi-Scale Template Matching)**：使用 OpenCV 对“看广告”、“关闭”、“确认”按钮进行高精度匹配。内置多尺度匹配策略，完美自适应不同设备分辨率（如 1280x720 与 1440x3200 屏幕）及广告页面不同的关闭按钮大小。
2.  **非极大值抑制 (NMS, Non-Maximum Suppression)**：内置 NMS 算法，对重叠的匹配候选框进行去重过滤，防止对同一个按钮多次重复点击。
3.  **智能置灰按钮跳过**：匹配到“观看广告”按钮后，会自动分析该区域内的 **“亮橙色”像素比例**。如果比例低于阈值，代表该按钮已次数用完变灰（如 0/5 次），脚本会自动跳过，避免无谓空点。
4.  **广告状态三合一轮询机制**：
    *   **信号 1 (像素差异)**：对比点击前后的界面截图像素差异比。
    *   **信号 2 (XML 节点数)**：监控 UI 层次结构 XML 节点数量的变化情况。
    *   **信号 3 (资源 ID 探测)**：探测常见广告联盟（穿山甲、优量汇等）特有的 `resource-id`。
    三项信号联合校验，精准判定广告是否成功打开。
5.  **安全危险区规避**：支持设置黑名单禁用区域（如抖音小游戏外壳的顶部菜单），防止误触退出游戏。
6.  **优雅退出**：监听 `Ctrl + C` 中断信号，退出时会自动清理并停止轮询，确保不会使设备处于异常状态。

## 🛠️ 环境依赖与安装

### 1. 安卓设备端配置
*   准备一台 Android 手机（或安装了游戏的安卓模拟器）。
*   进入 **设置 ➔ 开发者选项**：
    1.  开启 **USB 调试**。
    2.  开启 **USB 调试 (安全设置)** —— 即允许通过 USB 进行模拟点击（重要：小米/HyperOS 等设备若不开启此项，脚本只能截图而无法实现模拟点击）。
*   使用 USB 数据线将手机连接到电脑，确保在电脑终端运行 `adb devices` 时能看到您的设备。

### 2. 电脑端环境配置
在您的电脑终端，进入本项目目录并使用 pip 安装所需的 Python 依赖库：

```bash
pip install opencv-python numpy uiautomator2
```

> **提示**：首次使用 `uiautomator2` 时，可以在终端运行以下命令在连接的安卓设备上自动初始化并安装 `atx-agent` 守护进程：
> ```bash
> python3 -m uiautomator2 init
> ```

## 🏃‍♂️ 使用步骤

1.  **连接手机**：用数据线连接手机与电脑，确保 USB 调试已授权信任。
2.  **准备界面**：在手机上启动游戏，并进入到含有“观看广告/免费领取奖励”的活动列表页面。
3.  **运行脚本**：在电脑终端项目根目录下执行：
    ```bash
    python3 auto_ads.py
    ```
4.  **停止脚本**：若想在中途退出脚本，请在终端窗口直接按下 **`Ctrl + C`** 即可安全终止进程。
